Thank you for your interest in contributing to **mechAiLab**
mechAiLab is an open research platform dedicated to computational mechanics, structural materials, materials characterization and physics-guided ML.
Whether you are fixing documentation, improving code, adding datasets, or implementing mechanic models, your contributions are welcome.

**Research Philosophy**

Every contribution should support one or more of the following objectives:
Reproducible computational research
Physics-informed modelling
Open mechanics datasets
High quality documentation
Benchmarking classical mechanics and ML methods
Scientific transparency

Accuracy and reproducibility are always prioritized over model complexity

**Ways to Contribute**

You can contribute by:
Adding publicly available mechanics datasets
Implementing classical mechanics models
Developing ML baselines
Improving documentation
Creating tutorial notebooks
Fixing bugs
Improve code quality
Adding visualization tools
Improving benchmark pipelines

**Coding Guidelines**

Follow PEP 8 style guidelines
use descriptive variable names
Write clear docstrings
keep functions modular and reusable
Avoid duplicated code
Include type hints whenever practical

**Documentation**
Every new feature should include appropriate documentation
If application, update:
README.md
API documentation
Tutorial notebooks
Example scripts

Documentation is considered part of the contribution

**Datasets**
If contributing a dataset, please include:
Original publication
DOI (if available)
Source URL
License
Material system
testing conditions
Units
Metadata
Whenever possible, raw data should remain unchanged, while preprocessing should be performed.

**Benchmarks**
Benchmark implementations should:
Clearly describe evaluation metrics
Include baseline methods
Use fixed random seeds
Report software versions
Be reproducible from raw data

**Code of Conduct**
We are committed to maintaining a welcoming and respectful research environment.
Contributors are expected to:
Respect differing scientific viewpoints
Engage in constructive discussions
Credit original authors appropriately
Follow principles of academic integrity

**Citation**
If you use **mechAiLab** in academic research, please cite the repository using the citation information provided in the repository

**Questions**
For questions, suggestions, or collaboration opportunities, please open a GitHub Issue or start a GitHub Discussion.
We welcome contributions from researchers, students, and engineers interested in mechanics, structural materials and scientific ML.
